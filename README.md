Preguntados - Trabajo Individual Final.

Guía de descarga e instalación del proyecto:

1) Debe instalar Node.js, desde la página: https://nodejs.org/en/download, versión 16 o superior. 

2) Debe instalar un IDE como Visual Studio Code, o Webstorm, configurados para que entiendan los lenguajes Java Script, HTML, Java Script XML y CSS.

3) Debe descargar el código de este repositorio en su máquina local, levantarlo en su IDE favorito.

4) En la terminal dentro de su IDE, debe asegurarse de que esté en la ruta de directorios, en donde tiene descargado el proyecto, luego, debe ejecutar los siguientes comandos para la instalación de React y Vite:

npm install
npm run dev

5) Finalmente, debe abrir el link que le devuelve la terminal, en su navegador preferido, y comenzar a jugar!
